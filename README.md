🎬 CineVerse – Movie Recommendation System

📌 Project Overview

CineVerse is a **Movie Recommendation Web Application** built using **Python, Streamlit, TensorFlow, and Keras**.
The system recommends movies based on user input using **Deep Learning embedding techniques** and similarity matching.

This project demonstrates the use of **Machine Learning, NLP, and Web Deployment** in a real-time application.

---

## 🚀 Features

* 🎥 Movie recommendation based on similarity
* 🔎 Search movie by name
* 🤖 Deep Learning embedding model
* 🌐 Web interface using Streamlit
* ⚡ Fast and interactive UI
* ☁️ Deployed as live web app

## ⚙️ How It Works

1. Dataset of movies is loaded
2. Text / movie features are converted into embeddings
3. Neural network model learns movie similarity
4. User enters movie name
5. System finds closest matching movies
6. Recommended movies are displayed

---

## 🧩 Algorithm Used

* Embedding Layer (Deep Learning)
* Cosine Similarity
* Neural Network Model

Embedding layer converts movie data into vectors,
and similarity is calculated to recommend movies.
